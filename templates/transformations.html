Двухмерные трансформации свойство transform

Описание

Трансформирует элемент, в частности, позволяет его масштабировать, вращать, сдвигать, наклонять, а также комбинировать виды трансформаций.

Используется, как для создания динамического поведения объектов (например, при наведении мышки :hover), так и статических декоративных эффектов при вёрстке. Можно крутить, наклонять, перемещать, уменьшать и увеличивать объекты, изменять их форму.

Синтаксис

transform: <функция> [<функция>]* | none
transform: функция трансформации(значение трансформации)

Значения

функция
Функция трансформации.

none
Отменяет действие трансформации.

В CSS используется обратная система координат.

Её ключевой особенностью является то, что ось Y направлена вниз, а не вверх, так как веб-страница начинается с левого верхнего угла и идет вниз,

В связи с этим, положительные значения трансформации по оси Y переместят объект вертикально вниз. Для движения вверх используется отрицательное значение трансформации.

transform: translateX;

Сдвигает элемент по горизонтали на указанное значение. Положительное значение сдвигает вправо, отрицательное влево.Значение функции трансформации численное, а возможные единицы измерения — px, %, em или in.

transform: translateX(tx)

исходное положение

статическое изменение transform: translateX(50px);

статическое изменение transform: translateX(-50px);

динамическое изменение transform: translateX(50px);

динамическое изменение transform: translateX(-50px);

transform: translateY;

Сдвигает элемент по вертикали на указанное значение. Положительное значение сдвигает вниз, отрицательное вверх. Обратить внимание!!! Для движения вверх используется отрицательное значение трансформации.

transform: translateY(ty)

исходное положение

статическое изменение transform: translateY(20px);

статическое изменение transform: translateY(-20px);

динамическое изменение transform: translateY(20px);

динамическое изменение transform: translateY(-20px);

transform: translate();
transform: (translateY translateX);

Быстрый трюк с CSS: как центрировать объект точно по центру

Одновременно к объекту может быть применено несколько функций трансформации. При этом функции перечисляются через пробел после имени свойства transform, например:
transform: translateY(-100px) translateX(100px).
Такая трансформация переместит объект на 100 пикселей вправо и вверх по оси координат. Те же действия выполнит, объединяющая перемещение сразу по двум осям функция —
translate: transform: translate(перемещение по оси X [, перемещение по оси Y])
translate Сдвигает элемент на заданное значение по горизонтали и вертикали.

transform: translate(перемещение по оси X [, перемещение по оси Y])

Квадратные скобки указывают на то, что значение перемещения по оси Y является необязательным аргументом. В коде квадратные скобки писать не нужно, два значения перечисляются через запятую
transform: translate(50px, -50px);
причём у них могут быть разные единицы измерения. Если не указать значение перемещения по оси Y, а написать translate(перемещение по оси X), то значение перемещения по Y будет считаться равным 0 и функция будет работать аналогично translateX.

Эти функции работают одинаково:
transform: translate(100px);
transform: translate(100px, 0);
transform: translateX(100px);

В примере покажем одно и то же перемещение по горизонтали и вертикали, заданное двумя разными функциями
transform: translateX(50px) translateY(-45px); и
transform: translate(50px, -45px);

исходное положение

статическое изменение transform: translateX(50px) translateY(-50px);

статическое изменение transform: translate(50px, -45px);

динамическое изменение transform: translateX(50px) translateY(-45px);

динамическое изменение transform: translate(50px, -45px);

transform: scale();

Масштаб элемента по горизонтали и вертикали.

transform: scale(sx[, sy]);

Значение больше 1 увеличивает масштаб элемента, меньше 1 — уменьшает масштаб.

Вообще функция scale, как и translate, принимает 2 аргумента:

scale(масштаб-по-X [, масштаб-по-Y])

Если необязательный аргумент масштаб-по-Y не задан, то считается, что он такой же, как масштаб-по-X:

transform: scale(2) то же самое, что transform: scale(2, 2)

Рассмотрим примеры:
transform: scale(0.5); уменьшит объект в 2 раза;
transform: scale(2); увеличит объект в 2 раза;
transform: scale(0); полностью «схлопнет» объект, и его не будет видно;
transform: scale(1); оставит объект без изменений.

исходное положение

уменьшит объект в 2 раза, сожмёт к центру по осям; :hover {transform: scale(0.5);}
:active {transform: scale(2);}

увеличит объект в 2 раза во все стороны по осям; :hover {transform: scale(2);}
:active {transform: scale(0.5);}

полностью «схлопнет» объект, и его не будет видно; :hover {transform: scale(0);}
:active {transform: scale(2);}

оставит объект без изменений transform: scale(1);

Также в случае, когда необходимо масштабировать объект только по одной оси независимо от другой, существуют функции scaleX(масштаб-по-Х) и scaleY(масштаб-по-Y).

transform: scaleX;

Масштабирует элемент по горизонтали.

transform: scaleX(sx);

Значение больше 1 увеличивает масштаб элемента, меньше 1 — уменьшает масштаб.

Рассмотрим примеры:

transform: scaleX(0); полностью «схлопнет» объект, и его не будет видно;
transform: scaleX(1); оставит объект без изменений.

исходное положение

увеличит объект в 2 раза по горизонтали; :hover {transform: scaleX(2);}

полностью «схлопнет» объект, и его не будет видно; :hover {transform: scaleX(0);}
:active {transform: scaleX(2);}

оставит объект без изменений transform: scaleX(1);

transform: rotate;

Поворот элемента на заданный угол относительно точки трансформации, задаваемой свойством transform-origin.

transform: rotate(<угол>);

Положительное значение угла повернёт объект по часовой стрелке, а отрицательное — против. Единица измерения поворота — градусы deg, например: transform: rotate(180deg) повернёт объект на 180° по часовой стрелке, то есть перевернёт его.

Рассмотрим примеры:
transform: rotate(45deg); повернёт объект на 45° по часовой стрелке;
transform: rotate(90deg); повернёт объект на 90° по часовой стрелке;
transform: rotate(-45deg); повернёт объект на 45° против часовой стрелки;
transform: rotate(-90deg); повернёт объект на 90° против часовой стрелки.

исходное положение

повернёт объект на 45° (135°) по часовой стрелке; :hover {transform: rotate(45deg);}
:active {transform: rotate(135deg);}

повернёт объект на 90° (180°) по часовой стрелке; :hover { transform: rotate(90deg);}
:active { transform: rotate(180deg);}

повернёт объект на 45° (135°) против часовой стрелки; :hover { transform: rotate(-45deg);}
:active { transform: rotate(-135deg);}

повернёт объект на 90° (180°) против часовой стрелки. :hover { transform: rotate(-90deg);}
:active { transform: rotate(-180deg);}

При совместном применении трансформации поворота и перемещения есть небольшая тонкость. При повороте вместе с объектом на заданный угол поворачивается и вся его система координат:

transform: skewX;

Наклоняет элемент на заданный угол по горизонтали.

transform: skewX(<угол>)

исходное положение

статическое изменение transform: skewX(15deg);

статическое изменение transform: skewX(-15deg);

динамическое изменение transform: skewX(45deg);

динамическое изменение transform: skewY(-45deg);

transform: skewY;

Наклоняет элемент на заданный угол по вертикали.

transform: skewY(<угол>)

исходное положение

статическое изменение transform: skewY(15deg);

статическое изменение transform: skewY(-15deg);

динамическое изменение transform: skewY(45deg);

динамическое изменение transform: skewY(-45deg);

transform-origin

Устанавливает координаты точки, относительно которой будет происходить трансформация элемента.

Синтаксис transform-origin для трёхмерных трансформаций следующий:

transform-origin: (X) (Y) (X)

(X) Координата по оси X. Может принимать следующие значения:

| длина | проценты | left | center | right

Здесь длина — любая единица измерения CSS.

(Y) Координата по оси Y. Может принимать следующие значения:

| длина | проценты | top | center | bottom

(Z) Координата по оси Z. Может задаваться только в любых корректных единицах для измерения длины (исключая проценты).

Оси и значения показаны на рис.

Это свойство используется совместно со свойством transform и задаёт точку отсчёта системы координат, в рамках которой будет работать трансформация.

Синтаксис transform-origin для двухмерных трансформаций следующий:

transform-origin: точка-отсчёта-по-X [точка-отсчёта-по-Y]

Значения свойства задаётся в единицах измерения ширины в браузере (px, em...), в %, а также ключевыми словами left, right, top, bottom и center.

По умолчанию, значение transform-origin равно 50% 50%, то есть начало системы координат находится в центре объекта. Если не указывать значение точка-отсчёта-по-Y, то оно считается равным 50%.

исходное положение

начало системы координат находится в центре
transform: rotate(360deg);
transition: transform 1s ease;
transform-origin: по умолчанию.

начало системы координат внизу справа
transform: rotate(-360deg);
transition: transform 1s ease; transform-origin: 100% 50%

начало системы координат вверху справа
transform: rotate(-360deg);
transition: transform 1s ease; transform-origin: top right;

начало системы координат в центре правой стороны transform: rotate(360deg);
transition: transform 1s ease; transform-origin: center right;
